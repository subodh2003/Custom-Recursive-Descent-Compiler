# 🔧 Self-Derived Language → Assembly Language Translator

A **proprietary translator** developed in **C** that converts a **self-derived language** into **ARM64-style assembly**.  
This project showcases **compiler and interpreter design fundamentals**, including **lexical parsing**, **control flow translation**, and **assembly code generation**.

---

## 🚀 Features

✅ Translates a **custom minimalistic language** into **ARM64 assembly**  
✅ Supports:
- Function definitions and calls  
- Labels and branching  
- Conditional and unconditional jumps  
- Variable assignments and arithmetic operations  

✅ Demonstrates **core compiler construction principles** — parsing, symbol handling, and code generation  

---

## 🛠️ Getting Started

### **Prerequisites**
- GCC (or any C-compatible compiler)
- ARM64-compatible environment or emulator (e.g., **QEMU**) to run generated assembly

### **Compilation**
```bash
gcc saveme16.c -o translator
```

### **Running**
```bash
./translator < inputfile > 
```

- **inputfile:** your program written in the self-derived language (e.g., `input.txt`)  
- **outputfile:** generated ARM64 assembly output (e.g., `test.s`)

## 📚 Input Example

Below is an example source code written in the **self-derived language**:

```text
( $start 
  ( $r 1 
    X0 <- #0 
    X2 <- #0  
    :loop CB X0 == X1 :evn 
    CB X0 > X1 :odd 
    X0 +<- #2 
    b :loop 
    :odd X2 <- #0 
    :evn X2 <- #1 
    return   
  ) 
  ( $start 3 
    X1 <- #6 
    bl $r 
  ) 
)
```

## ⚙️ Output Example

The corresponding **ARM64 assembly** generated by the translator:

```asm
.section .text
.global _start

__sbr_r:
    str FP,[SP,#-8]!
    str LR,[SP,#-8]!
    mov FP,SP
    mov X0,#0
    mov X2,#0
__r_loop:
    cmp X0,X1
    beq  __r_evn
    cmp X0,X1
    bgt  __r_odd
    add X0,X0,#2
    b __r_loop
__r_odd:
    mov X2,#0
__r_evn:
    mov X2,#1
    ret
    bl __ret_ret

__sbr_start:
    str FP,[SP,#-8]!
    str LR,[SP,#-8]!
    mov FP,SP
    mov X1,#6
    bl __sbr_r
    bl __ret_ret

_start:
    mov X0, #1
    bl __sbr_start
    mov W8, #93
    svc #0
    ret

__ret_ret:
    mov SP,FP
    ldr LR,[SP],#8
    ldr FP,[SP],#8
    ret
```

## 🧩 Explanation

| Concept | Self-Derived Syntax | Translated Assembly |
|----------|--------------------|---------------------|
| **Function Definition** | `$r` | `__sbr_r:` |
| **Variable Assignment** | `X0 <- #0` | `mov X0,#0` |
| **Label Declaration** | `:loop` | `__r_loop:` |
| **Conditional Branch** | `CB X0 == X1 :evn` | `cmp X0,X1` + `beq __r_evn` |
| **Loop / Branch** | `b :loop` | `b __r_loop` |
| **Function Call** | `bl $r` | `bl __sbr_r` |
