# 🔧 Self-Derived Language to Assembly Language Translator

This is a **proprietary** translator developed in C that converts a **self-derived language** into **assembly language** (ARM64-style). It demonstrates core concepts of compilers and interpreters, including parsing, control flow translation, and code generation.

---

## 🚀 Features

✅ Translate a custom, minimalistic language into assembly  
✅ Supports:
- Function definitions
- Labels and branching
- Conditional and unconditional jumps
- Variable assignments
- Function calls
✅ Demonstrates basic compiler design principles

---

## 🛠️ Getting Started

### Prerequisites

- GCC (or compatible C compiler)
- ARM64-compatible environment or an emulator (QEMU) to run the generated assembly

📚 Input Example
Below is an example input file written in the self-derived language:

ruby
Copy code
( $start 
 ( $r 1 
    X0 <- #0 
    X2 <- #0  
    :loop CB X0 == X1 :evn 
    CB X0 > X1 :odd 
    X0 +<- #2 
    b :loop 
    :odd X2 <- #0 
    :evn X2 <- #1 
    return   
 ) 
  ( $start 3 
    X1 <- #6 
    bl $r 
 ) 
 )

⚙️ Output Example
Here’s the translated ARM64 assembly generated by the translator:

assembly
Copy code
.section .text
.global _start

__sbr_r:
    str FP,[SP,#-8]!
    str LR,[SP,#-8]!
    mov FP,SP
    mov X0,#0
    mov X2,#0
__r_loop:
    cmp X0,X1
    beq  __r_evn
    cmp X0,X1
    bgt  __r_odd
    add X0,X0,#2
    b __r_loop
__r_odd:
    mov X2,#0
__r_evn:
    mov X2,#1
    ret
    bl __ret_ret

__sbr_start:
    str FP,[SP,#-8]!
    str LR,[SP,#-8]!
    mov FP,SP
    mov X1,#6
    bl __sbr_r
    bl __ret_ret

_start:
    mov X0, #1
    bl __sbr_start
    mov W8, #93
    svc #0
    ret

__ret_ret:
    mov SP,FP
    ldr LR,[SP],#8
    ldr FP,[SP],#8
    ret


📝 Explanation
Functions: $r translates to __sbr_r; $start to __sbr_start.
Variable Assignments: X0 <- #0 → mov X0,#0.
Labels: :loop → __r_loop.
Conditional Branches: CB becomes cmp + beq or bgt.
Loops: b :loop → b __r_loop.
Function Calls: bl $r → bl __sbr_r.
